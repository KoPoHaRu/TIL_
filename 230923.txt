<ul> 순서가 없는 리스트
나열된 항목들이 순서와 상관없을때 사용하는 리스트
<ol> 순서가 있는 리스트
나열된 항목들이 순서와 관련있을때 사용되는 리스트
<dl>용어를 설명하는 리스트
<li> x <dt>용어 <dd>정의, 설명
<dd>코드는 중복사용 가능
리스트의 중첩
 <ol>태그와 <li>태그는 중첩
<ol>태그안에는 <li>태그만 사용가능
<li>태그를 사용하면 부모태그는 <ol>태그와 <ul>태그만 사용가능
<li>태그에는 자식으로 여러가지 태그 사용가능
 
이미지 태그
<img>문서에 이미지를 삽입하는 태그
src = 이미지 경로를 지정함 상대 경로와 절대 경로가 존재
상대경로는 현재 웹문서에 페이지를 기준으로 상대적으로 이미지의 위치를 나타내는 경로
절대경로는 실제 그 이미지가 위치해있는곳에 전체있는곳
상대경로를 사용할때는 ./를 사용 생략도 가능
../은 상위폴더로 이동하는 명령어
현재 폴더에서 사용하려면 folder/ 명령어를 사용
alt = 이미지의 대체 텍스트를 입력함
width / height = 이미지의 크기를 지정함
하나라도 크기를 조정하면 나머지 속성도 자동으로 맞춰줌
gif : 색이 제한적이지만 용량이 작고 애니메이션과 투명 이미지가 가능
jpg : 자연스러운 색상 표현이 가능하고 압축률이 높음 사진이나 일반적인 그림에 사용 투명지원은 x
png : jpg와 비교했을때 이미지 손실이 없고 높은 퀄리티의 이미지, 투명과 반투명 모두 지원함

표
<td>,<th> 셀을 표현해주는 태그
<tr>:<td>,<th>태그를 감싸주는 태그
<table>:위에 태그들을 감싸주는 태그
<caption> : 표의 제목 가장 위에서 선언되어야함
<thead> : 제목 행을 그룹
<tfoot> : 바닥 행을 그룹
<tbody> : 본문 행을 그룹

rowspan : 셀을 세로방향으로 병합
